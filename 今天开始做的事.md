# 今天就开始做的事（6 月 16 日）

按顺序来，做完一项勾一项。

---

## ☐ 1. 安装 Go 环境（30 分钟）

1. 打开 https://go.dev/dl/
2. 下载 Windows 安装包（go1.21+ 或最新版）
3. 安装，全部默认
4. 打开命令行（PowerShell），输入：
   ```
   go version
   ```
   看到 `go version go1.xx.x windows/amd64` 就成功了

---

## ☐ 2. 拉取项目依赖并跑起来（10 分钟）

```bash
cd "C:\Users\Administrator\Desktop\个人\campus-recruit"
go mod tidy
go run main.go
```

打开浏览器访问 http://localhost:8080/ping，看到 `{"msg":"pong"}` 就成功了。

---

## ☐ 3. 注册账号（15 分钟）

- **GitHub**：如果还没有，注册 github.com
- **牛客网**：nowcoder.com，看面经用
- **LeetCode**：leetcode.cn 或 leetcode.com，刷题用

## ☐ 4. 开始看 Go 教程（至少 1 小时）

打开 [A Tour of Go](https://go.dev/tour/)，跟着在线交互式教程走一遍：

1. Welcome 到 Methods 部分，全部在网页上敲一遍
2. 重点关注：变量声明、切片、map、struct、函数、指针
